## Overview

This repository presents a minimal structural framework in which key physical quantities emerge from fixed ratios rather than parameter fitting.

The framework is based on two structural constants:

- `R = 13/6`
- `S = 31/24`

These constants are used across cosmological density relations, electromagnetic coupling, mass hierarchy relations, gravity-sector relations, and quark-mass hierarchy calculations.

### Cosmology

The density parameters are generated from the same structural ratio \(R\):

```math
\Omega_b = \frac{1}{3R^2+3R},
\qquad
\Omega_m = 3R\Omega_b,
\qquad
\Omega_\Lambda = 3R^2\Omega_b,
\qquad
\Omega_{dm} = \Omega_m - \Omega_b.
```

These relations imply the compact consistency relation

```math
\Omega_m^2 = 3\Omega_\Lambda\Omega_b.
```

### Quark Masses

The quark masses are generated by the unified relation

```math
\frac{m_q}{m_e}
=
K_q(3R^2S)
\left(
\frac{8\pi^{u_q}R^{u_q+1}}{S}
\right)^n .
```

The structural coefficients are

```math
K_q =
\left\{
\frac{1}{\Psi_u},
\frac{1}{\Psi_d},
1+\frac{1}{\Psi_d},
1-\frac{1}{2\Psi_d},
2+\frac{1}{\Psi_u},
2+\frac{1}{\Psi_d}
\right\},
\qquad
(q=u,d,c,s,t,b).
```

with

```math
\Psi_u = 2R,
\qquad
\Psi_d = 2.
```

The selector \(u_q\) separates up-type and down-type sectors:

```math
u_q =
\begin{cases}
1 & (u,c,t), \\
0 & (d,s,b).
\end{cases}
```

The generation index is

```math
n =
\begin{cases}
0 & (u,d), \\
1 & (c,s), \\
2 & (t,b).
\end{cases}
```

## Notes on Interpretation

The numerical relations in this repository are presented as structural correspondences between fixed dimensionless ratios and observed physical quantities.

They are not introduced as fitted empirical formulas.

For quantities with scheme dependence, such as the top-quark mass, the comparison should be interpreted as agreement with the relevant observed mass scale rather than tuning to a single central value.
